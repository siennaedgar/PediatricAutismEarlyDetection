# Early Detection of Pediatric Autism Using Machine Learning

![Project Banner](https://medschool.duke.edu/sites/default/files/styles/social_media_card/public/2021-08/mdm_0086_72dpi.jpg?h=de0075d1&itok=bJZZGueb)

## Overview
This project focuses on the early detection of Autism Spectrum Disorder (ASD) in children using machine learning techniques. Early identification of ASD can significantly improve intervention outcomes, making predictive models a valuable tool in pediatric healthcare.

The model analyzes behavioral, developmental, and/or clinical data to identify patterns associated with autism risk.

---

## Objectives
- Develop a machine learning model to predict likelihood of ASD in pediatric populations  
- Improve early screening accuracy compared to traditional methods  
- Identify key predictive features associated with autism  
- Support healthcare professionals with data-driven insights
